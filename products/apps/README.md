# Applications

Purpose: suite-level helper applications that are not `.scr` savers.
Belongs here: the real `suite` control app, BenchLab, SDK-adjacent helper apps if later justified, and narrowly scoped legacy stubs that mark retired app names.
Does not belong here: saver product code or shared platform internals.
Status: `benchlab` and `suite` are real companion apps. `player` is only a superseded legacy stub.
Channel rule: app products stay secondary to the primary Core saver ZIP and do not redefine the standalone saver line.
Type: product runtime space.
