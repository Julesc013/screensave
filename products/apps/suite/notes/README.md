# Suite Notes

Purpose: stable notes and role-boundary reminders for the `suite` app.

- `suite` is the end-user browser, launcher, preview, and configuration surface.
- Standalone `.scr` savers remain first-class products.
- `anthology` remains the suite meta-saver for cross-saver randomization.
- BenchLab remains the diagnostics harness.

Do not turn this directory into a marketplace, pack editor, or SDK workspace.
