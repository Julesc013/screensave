# Plasma v2 RC1

Status: release candidate, not stable.

`plasma-v2-rc1` is the first Plasma v2 release candidate. It preserves the
reference-preview profile `plasma.v2.reference.preview` and keeps stable
promotion blocked.

Included candidate evidence:

- Release-candidate contract.
- Non-published package stage.
- Reference proof and release-candidate proof matrix.
- GL11 candidate comparison with software-reference fallback.
- Performance envelope with warnings.
- Support and known-limits wording.
- Manager preview and Workbench release-readiness review.
- Artistic decision accepted for release-candidate preparation.

Not included:

- Stable release.
- Release publication.
- Compatibility certification.
- Public SDK stability.
- All-saver migration.

The next product question is stable promotion, not broader feature expansion.
