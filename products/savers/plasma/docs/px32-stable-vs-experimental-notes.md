- The new subset is real, but it is still bounded and premium-aware.
- The current default stable path remains Classic-first, lower-band-first, raster, and flat.
- Fresh local compiled reruns remain blocked in this checkout, so stable widening would overclaim current proof.
- `bounded_billboard_volume` and broader scenic dimensional breadth remain unsupported.
