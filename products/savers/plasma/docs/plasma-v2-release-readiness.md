# Plasma v2 Release Readiness

Status: release-readiness contract, not release.

Plasma v2 is currently a stable candidate. The release-readiness track checks
whether the candidate can be packaged, previewed, inspected, reviewed, and
handed to a later release-candidate decision without hiding gaps.

Release readiness keeps these facts separate:

- proof is not visual acceptance,
- visual acceptance is not stable release,
- package staging is not publication,
- artifact audit is not compatibility certification,
- release-candidate preparation is not stable promotion.

The release-readiness gate must cover reference proof, acceleration comparison,
performance envelope, Pack v1 candidate data, Workbench release-readiness facts,
Manager pack preview, artifact staging, known limits, support wording, visual
review, final artistic decision packet, and release-promotion status.

For PAW-G, the target state is `release-readiness-reviewed` with `stable =
false` and release promotion still blocked.
