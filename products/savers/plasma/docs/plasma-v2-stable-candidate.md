# Plasma v2 Stable Candidate

Status: stable-candidate, not stable.

Plasma v2 now has a reviewed, proof-backed, pack-authored stable-candidate
slice. The product-owned software reference remains canonical; the first GL11
candidate is admitted only with explicit software-reference fallback evidence.

The candidate includes:

- `plasma.v2.reference.preview` exact proof captures,
- bounded Pack v1 candidate metadata,
- deterministic VisualIntent candidates,
- material and control proof matrices,
- reference-versus-GL11-candidate comparison evidence,
- a performance/resource envelope with portable-runner limits,
- Workbench Author, Inspect, Compare, Profile, and Review facts,
- development-plane reference-only evidence projection.

This state does not make Plasma v2 stable. Final artistic acceptance, stable
release promotion, compatibility certification, Manager packaging, broader
renderer lanes, and public SDK stability remain blocked for a later
release-readiness wave.
