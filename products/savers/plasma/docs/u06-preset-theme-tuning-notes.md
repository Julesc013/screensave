## Short Summary

`U06` does four things:

- retunes the surviving stable and experimental preset catalogue so it is less
  crowded and more intentionally distinct
- retunes every surviving built-in theme so the warm, cool, monochrome, and
  dark-room identities separate more clearly
- strengthens the default `plasma_lava` path without changing its preserved key
  or stable-lane posture
- calms preview, long-run, and transition timing so the product is more
  supportable before the later proof-harness and QA phases

This note remains in place so older `PX50` and corrective-series references do
not break, but the new U06 doc set above is the current source of truth.
