# Observatory

Observatory is the celestial exhibit saver in the Places and Atmosphere Family.

It provides:

- curated orrery, chart-room, and dome-watch studies with ordered celestial motion
- modest speed and detail controls instead of an astronomy package or ephemeris system
- brass-observatory, monochrome-chart-room, deep-blue-dome, eclipse-watch, planetarium-blue, brass-eclipse, and quiet-museum themes and presets
- calmer exhibit pacing and periodic body or star refresh so long runs stay serene and legible
- deterministic startup when requested and preview-safe long-run behavior through the shared saver and renderer contracts

Observatory stays intentionally bounded.
It is not a star catalog, a real astronomy library, or a heavyweight orbital simulator.
