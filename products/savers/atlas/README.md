# Atlas

Atlas is the curated fractal-voyage saver in the Heavyweight Worlds Family.

It provides:

- curated Mandelbrot voyage, atlas, and Julia-study presentations instead of a full exploration lab
- staged bitmap refinement so the image settles in progressively without requiring a modern renderer tier
- modest speed and refinement controls, deterministic starting routes, and preview-safe sizing
- atlas-monochrome, liquid-nebula, terminal-mathematics, deep-ultraviolet, nebula-voyage, mathematics-plate, and museum-print-room themes and presets
- more deliberate route progression so anthology-style long runs feel curated without becoming a formula editor

Atlas stays intentionally bounded.
It is not an arbitrary formula parser, a research workbench, or a broad fractal engine.
