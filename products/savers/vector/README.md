# Vector

Vector is the abstract software-3D product in the Heavyweight Worlds Family.

It provides:

- three related scene families: wireframe structure field, tunnel/ribbon flow, and line terrain or canyon flyovers
- a product-local projection and camera model built for restrained geometric motion
- modest scene, speed, and detail controls instead of a broad 3D sandbox
- curated themes and presets tuned for both the mandatory GDI path and optional GL11
- preview-safe and long-run-stable behavior through the shared saver and renderer contracts

Vector stays intentionally bounded.
It is not a general 3D engine, scene editor, or model viewer.
