# Template Pack Example

This is the minimal SDK example pack.

It demonstrates:

- a real `pack.ini`
- one preset export/import file
- one theme export/import file
- safe relative pack entries

The example targets `template_saver` so it lines up with the real saver template.
When you copy it into a real saver tree, replace `template_saver`, `starter_showcase`, and the display text in one pass.
