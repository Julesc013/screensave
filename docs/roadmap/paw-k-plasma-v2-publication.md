# PAW-K Plasma v2 Publication

Status: planned next gate after PAW-J publication prep.

PAW-K may publish Plasma v2 only after the PAW-J publication-ready packet is
accepted, PAW-K0 hardware visual proof is passed or explicitly deferred by the
project owner, and the publication gate verifies final release archive names,
release notes, checksums, provenance, public links, release-page truth, and a
publication receipt.

PAW-K must still preserve these exclusions unless a later reviewed gate changes
them explicitly:

- no compatibility certification broadening
- no all Core saver migration
- no public SDK freeze
- no Manager install/apply mutation beyond its own gate
- no graphical Workbench MVP unless PAW-W1 is opened separately
- no macOS/Linux host claim
- no runtime executable plugin ecosystem

PAW-J created a local publication-prep packet only. PAW-K is the earliest gate
allowed to perform public release publication.

PAW-K0 is the recommended pre-publication closeout gate. It records real-display
fullscreen proof, renderer-tier comparison, frame-time or pacing notes, and the
human visual note. PAW-K must not treat publication-ready metadata alone as a
substitute for that operator-owned proof unless the project owner records a
deliberate deferral.
