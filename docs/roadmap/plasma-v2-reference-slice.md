# Plasma v2 Reference Slice

Status: reference-preview candidate.

Opened on: 2026-06-26.

Gate C source of authority: `tools/scripts/check_gate_c_acceptance.py`.

AIDE work-unit packet: `.aide/work_units/plasma-v2-reference-slice.toml`.

## Purpose

PAW-DX makes Plasma the first post-Gate-C product slice over the accepted
portable v2 seam. The goal is one bounded, proof-backed Plasma v2 reference
preview slice, not a broad rewrite, release promotion, or universal graphics
graph.

The governing doctrine stays fixed:

```text
Portable meaning.
Native delivery.
Deterministic proof.
Optional automation.
```

## First Slice Scope

- product-owned `plasma_spec_v2`
- deterministic U09 migration table
- classic interference reference path
- one radial or warped generator
- domain warp and bounded feedback
- continuous, banded, and contour output
- `plasma_lava`, `aurora_cool`, `oceanic_blue`, `museum_phosphor`, and `quiet_darkroom` materials
- flat presentation
- restrained CRT/dither treatment
- software reference proof profile `plasma.v2.reference.preview`
- data-only `packc` skeleton for bounded Plasma specs
- minimal Workbench Inspect/Author support through the same proof path as CI

## Boundaries

Do not use this opening to start a full Plasma universe.

This slice must not:

- migrate all savers,
- introduce a public SDK declaration,
- admit generic AIDE command execution,
- certify compatibility,
- promote a release,
- treat automated checks as artistic acceptance,
- turn VisualIntent into a universal executable graph.

Portable v2 Gate C proves the Nocturne/Ricochet canary seam. Plasma v2 must now
earn its own product evidence one bounded slice at a time.

## Exit Definition

PAW-DX exits when Plasma v2 has a bounded product spec, a deterministic U09
migration map, a software reference path, material Basic controls, reference
captures, a proof bundle, a data-only `packc` example, Workbench
Author/Inspect support, AIDE EvidencePackets, and a preview gate. The exit
state is `reference-preview`, not stable.
