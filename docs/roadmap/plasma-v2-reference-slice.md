# Plasma v2 Reference Slice

Status: active program opened by portable v2 Gate C acceptance.

Opened on: 2026-06-26.

Gate C source of authority: `tools/scripts/check_gate_c_acceptance.py`.

## Purpose

PAW-D makes Plasma the first post-Gate-C product slice over the accepted
portable v2 seam. The goal is a bounded, proof-backed Plasma reference slice,
not a broad rewrite or a universal graphics graph.

## First Slice Scope

- `plasma_spec_v2`
- U09 migration table
- classic interference reference path
- one radial or warped generator
- domain warp
- bounded feedback
- continuous, banded, and contour output
- four materials
- flat presentation
- restrained CRT/dither treatment
- software reference proof
- one accelerated realization only if evidence-backed
- minimal Workbench Inspect/Author support

## Boundaries

Do not use this opening to start a full Plasma universe.

This slice must not:

- migrate all savers,
- introduce a public SDK declaration,
- admit generic AIDE command execution,
- certify compatibility,
- promote a release,
- treat automated checks as artistic acceptance,
- turn VisualIntent into a universal executable graph.

Portable v2 Gate C proves the Nocturne/Ricochet canary seam. Plasma v2 must now
earn its own product evidence one bounded slice at a time.
