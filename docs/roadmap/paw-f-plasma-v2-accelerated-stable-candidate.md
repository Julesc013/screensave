# PAW-F Plasma v2 Accelerated Stable Candidate

Status: active stable-candidate evidence wave.

Source milestone: `plasma-v2-reviewed-preview`.

PAW-F moves Plasma v2 from reviewed preview to stable candidate. It does not
publish a release, certify compatibility, freeze a public SDK, or make final
artistic acceptance.

## Objective

The wave admits one narrow acceleration candidate, compares it with the
software reference oracle, records performance and resource envelopes, hardens
bounded data packs, extends Workbench with Profile and Review facts, and keeps
AIDE as a reference-only evidence projection.

## First Acceleration Candidate

The first candidate is the GL11 realization lane because U09 already preserved
`gdi` as the mandatory floor and `gl11` as the preferred lower-band path.

The software reference remains canonical. If GL11 is unavailable or not active
in the portable proof runner, the candidate must degrade honestly to the
software reference and record that fallback in evidence.

## Claim Boundary

Stable candidate means the product has enough proof, pack, performance,
comparison, and review evidence to enter release-readiness work. It is not:

- stable release,
- final artistic acceptance,
- compatibility certification,
- all-saver migration,
- executable pack admission,
- automatic AIDE mutation,
- automatic product promotion.

The next responsible track is `plasma-v2-release-readiness`.
