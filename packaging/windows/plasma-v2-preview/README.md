# Plasma v2 Preview Package Stage

Status: staged preview package, not published.

This directory records a release-readiness package stage for the Plasma v2
stable candidate. It references the current Windows `.scr` build output, the
bounded Pack v1 candidate example, proof bundles, evidence indexes, known
limits, and checksums.

No binary build output is committed here. The `.scr` path in `manifest.toml`
is an output reference that must be produced by the build controller.

This package stage is not stable release, final artistic acceptance,
compatibility certification, installer admission, Manager installation, or
public SDK stability.
