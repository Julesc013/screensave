# Plasma v2 Stable-Promotion Preview Notes

Plasma v2 is staged for a stable-promotion decision from `plasma-v2-rc1`.

Included:

- Portable v2 Plasma reference profile.
- Current Windows x86 `.scr` artifact reference.
- Packc v1-candidate data-only pack path.
- Manager preview and Workbench release-readiness refs.
- Stable-promotion proof, support, provenance, and review refs.

Not included:

- Public publication.
- Compatibility certification.
- Public SDK stability.
- All-saver migration.
- Runtime executable plugin admission.
