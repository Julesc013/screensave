# AIDE Controller

This directory contains ScreenSave-local AIDE Lite controller metadata and
generated advisory outputs.

Tracked source files:

- `failure-taxonomy.yaml` defines the deterministic failure classes accepted by
  the imported AIDE Lite outcome controller.

Generated files such as `outcome-ledger.jsonl`, `latest-outcome-report.md`, and
`latest-recommendations.md` are local operation outputs. They are not product
truth, do not mutate source, and do not promote release or compatibility claims.
