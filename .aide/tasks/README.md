# AIDE Task OS Schemas

This directory contains Task OS v0 JSON Schemas for report-only WorkUnit records, attempts, blockers, repairs, waves, and checkpoints.

X-OS-00 defines contracts only. X-OS-01 adds report-only inspection/planning commands over those contracts. Neither phase implements scheduling, repair apply, branch creation, merges, pushes, checkpoint apply, promotion, provider/model calls, network calls, release publication, or target-repo mutation.

The schemas are portable surfaces for X-OS-01 report-only commands and later capability reality work in X-OS-02.
