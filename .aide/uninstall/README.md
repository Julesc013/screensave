# AIDE Uninstall Model v0

Q46 defines uninstall as observation, ownership classification, no-apply
planning, dry-run reporting, and validation only.

Uninstall is not blanket `.aide` deletion. Future removal candidates are limited
to evidence-backed AIDE-owned portable files, and even those remain no-apply in
Q46. Target memory, queue, evidence, golden tasks, generated target state,
manual content, existing tools, local state, and unknown-ownership files are
preserved or require manual review.
