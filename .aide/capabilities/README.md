# AIDE Capability Reality Seeds

This directory contains source-side, report-only capability seed records for X-OS-02.

The seed records are not product promises. They are bounded inputs for deterministic AIDE Lite scans that classify current repository evidence as planned, specified, stubbed, implemented, tested, exposed, documented, deprecated, removed, or unknown. Generated reports live under `.aide/reports/capability-*`.

The scanner must keep docs-only, fixture-only, report-only, dry-run-only, no-call, release-draft, source-generated, and target-specific evidence below live implementation or publication claims unless independent runtime evidence exists.
