# Evidence Packets

This directory is reserved for AIDE-facing pointers to proof bundles and
validation captures.

The first pilot records the current PE audit at:

- [`../../validation/captures/truth-proof-baseline/pe-audit-current.txt`](../../validation/captures/truth-proof-baseline/pe-audit-current.txt)

Evidence packets may point at repository evidence, but they must not invent
passing results or overwrite validator output.
