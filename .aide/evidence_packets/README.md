validation captures.

- [`../../validation/captures/truth-proof-baseline/pe-audit-current.txt`](../../validation/captures/truth-proof-baseline/pe-audit-current.txt)

- [`screensave-fixed-command-bridge.toml`](screensave-fixed-command-bridge.toml)

passing results or overwrite validator output.
