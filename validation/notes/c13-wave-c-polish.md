# C13 Wave C Polish Note

This note records the bounded final saver-specific polish pass that followed the `C12` suite quality bar.

## Scope

Wave C covered:

- `stormglass`
- `transit`
- `observatory`
- `vector`
- `explorer`
- `city`
- `atlas`
- `gallery`
- `anthology`

Wave C did not broaden into:

- out-of-wave saver families
- packaging or installer changes
- `suite` app feature expansion
- new renderer tiers or broad renderer-capability work

## Product Outcomes

- `stormglass` now has calmer atmospheric event timing, stronger winter-streetlamp and monochrome-cold-pane variants, and better preview-safe long-run refresh.
- `transit` now has calmer cinematic light reseeding, stronger motorway-fog and harbor-midnight variants, and better preview-safe route pacing.
- `observatory` now has calmer exhibit pacing, stronger planetarium-blue and brass-eclipse variants, and clearer long-run body and star refresh.
- `vector` now has stronger amber-wire and terrain-glide curation, better preview-aware scene reseeding, and calmer long-run compositional refresh.
- `explorer` now has calmer deterministic route refresh, stronger eerie-corridor and amber-service variants, and better preview-safe traversal pacing.
- `city` now has stronger harbor and rooftop variants, calmer long-run light and fog choreography, and preview-aware scenic reset behavior.
- `atlas` now has stronger nebula-voyage and mathematics-plate curation, calmer preview hold timing, and more deliberate route progression.
- `gallery` now has stronger amber-compatibility and GL21-prism-hall curation plus tier-change-aware scenic refresh.
- `anthology` now has calmer preview-safe interval defaults, stronger compatibility and scenic presets, and anti-repeat selection damping for better long-run anthology behavior.

## Shared Surface

- The Wave C saver set now has stronger preset and theme coverage without widening the shared settings surface.
- `stormglass`, `observatory`, `explorer`, `city`, and `gallery` now expose the real preset/theme/randomization import-export hooks they advertise through the shared saver contract.
- Preview-safe resize paths now refresh theme and session state more coherently across the scenic and showcase products that most needed it.
- `anthology` remains bounded: it still orchestrates inner savers in-process and now avoids some direct repetition without becoming a larger suite engine.

## Validation Level

- Validation for `C13` Wave C is static and source-level in this environment.
- No supported MSVC or MinGW toolchain was available here for a fresh native build or runtime capture pass.
- The Wave C smoke tests and the `check_wave_c_surface.py` validator are intended to keep the repo surface honest until final rerelease hardening runs.
